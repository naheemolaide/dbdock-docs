## Quick Start

Get up and running with DBDock in seconds.

```bash
npx dbdock init      # Interactive setup
npx dbdock backup    # Create backup
npx dbdock restore   # Restore backup
npx dbdock status    # View schedules and service status
```

## Installation

Install the CLI globally or use it directly with `npx`.

```bash
npm install -g dbdock
```
